🇨🇳⭐️⭐️⭐️⭐️⭐️ 专注于前端领域，“凡事必有回响”。⭐️⭐️⭐️⭐️⭐️ 🇨🇳

技术文档「它」的核心要素应该包括：
1. 它是什么？
2. 它有什么作用？换句话说，就是它能帮我们解决什么问题？
3. 它的优势是什么？有什么不足？
4. 它的竞品有哪些？各有什么特点？
5. 我要怎么用它？需要注意什么地方？
:::

::: tip 直接运行ts文件
ts-node 可以直接运行ts代码, 需要在全局去安装。
:::
```js
// 1. 全局安装 typescript
$ npm i -g typescript
// 2. 编译TS文件： tsc **.ts

// 1. 全局安装 ts-node
$ npm i -g ts-node
// 2. 运行TS文件： ts-node **.ts
```
